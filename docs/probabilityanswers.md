# Réponses au Mandat en Cas de Catastrophe

- Années nécessitant une action selon la Règle 1 : 2017

- Années nécessitant une action selon la Règle 2 : 2019, 2017

- Nombre d’années pour la Règle 1 : 1

- Nombre d’années pour la Règle 2 : 2

- Fraction pour la Règle 1 : 1/5

- Fraction pour la Règle 2 : 2/5

- Pourcentage pour la Règle 1 : 20 %

- Pourcentage pour la Règle 2 : 40 %

- Et si vous n’aviez pas l’IPC :  
Vous pourriez identifier le nombre d’années de sécheresse durant lesquelles vous auriez souhaité ou auriez eu besoin d’agir au cours des dernières années. Si vous êtes capable de classer les années selon leur importance pour intervenir, vous seriez en mesure de décider quoi faire lorsque votre capacité d’action est limitée.
