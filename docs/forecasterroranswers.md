# Réponses – Erreur de Prévision

- Pourriez‑vous utiliser la prévision pour agir plusieurs mois avant le début de la sécheresse ?  
	Réponse : Oui

- Cela vous donnerait‑il plus de temps pour vous préparer ?  
	Réponse : Oui

- Cela pourrait‑il réduire les impacts négatifs de la sécheresse sur les populations ?  
	Réponse : Oui

- Quel pourrait être le problème en utilisant la prévision ?  
	Réponse : Elle peut être erronée

- Quelle année est le pire événement humanitaire dans le tableau ?  
	Réponse : 2017

- La prévision d’avril s’est‑elle déclenchée cette année‑là ?  
	Réponse : Oui, c’est ce qu’on appelle une action justifiée

- Quel est le deuxième pire événement humanitaire dans le tableau ?  
	Réponse : 2019

- La prévision d’avril s’est‑elle déclenchée cette année‑là ?  
	Réponse : Non, c’est ce qu’on appelle un échec à agir

- Quel est le troisième pire événement humanitaire ?  
	Réponse : 2015

- Cette année compte‑t‑elle comme une année où vous auriez dû agir selon la Règle 2 ?  
	Réponse : Non

- La prévision se déclenche‑t‑elle cette année‑là ?  
	Réponse : Oui, c’est ce qu’on appelle une action en vain

- Que pensez‑vous du fait d’agir accidentellement une année mauvaise, mais pas assez mauvaise pour exiger une action selon le mandat officiel ?  
	Réponse : Vous êtes le mieux placé pour en juger

- Augmenter la fréquence pourrait‑il éviter les échecs à agir ?  
	Réponse : Oui

- Augmenter la fréquence pourrait‑il entraîner davantage d’actions en vain ?  
	Réponse : Oui

- Le nombre de crises humanitaires influence‑t‑il la fréquence d’action ?  
	Réponse : Oui

- Le coût de l’action et le budget influencent‑ils la fréquence d’action ?  
	Réponse : Oui

- La précision des prévisions influence‑t‑elle la fréquence d’action ?  
	Réponse : Oui

- Les acteurs humanitaires, les gestionnaires de financement et les scientifiques des prévisions doivent‑ils travailler ensemble pour déterminer les meilleurs seuils de déclenchement ?  
	Réponse : Oui

- L’un de ces experts peut‑il déterminer correctement les seuils sans consulter les autres ?  
	Réponse : Non
