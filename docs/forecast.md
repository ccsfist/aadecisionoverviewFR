# Prévisions 

![](yearforecasttable.png)

Imaginez maintenant que le service météorologique puisse fournir, dès le mois d’avril, une prévision indiquant si la saison agricole de juillet‑août‑septembre risque d’être marquée par une sécheresse.

Vous pouvez consulter le tableau ci‑dessus pour voir un exemple réel de ce type de prévision.
La colonne « prévision » indique la probabilité de sécheresse : plus le chiffre est élevé, plus la probabilité est forte.

Nous pouvons inverser le raisonnement utilisé pour décider d’agir et voir comment exploiter la prévision.
Les questions ci‑dessous vous guideront à travers ce processus.
