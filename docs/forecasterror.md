# Erreur de Prévision

![](yearforecasthumantable.png)

Les événements humanitaires sont maintenant indiqués dans le tableau, à côté des prévisions.
Ils sont présentés sous forme de classement :
l’année ayant le plus grand nombre de personnes en insécurité alimentaire est classée 1,
la deuxième 2, et ainsi de suite.

Ce tableau n’est qu’un extrait d’un tableau beaucoup plus long que vous verrez dans les étapes suivantes. Le tableau complet couvre la période 2000–2024, en classant les pires années.
Les pires années — de la plus grave à la cinquième — ne se situent pas toutes entre 2015 et 2019.
Ainsi, dans ce tableau, la pire année commence avec le rang 6.
